## ğŸ§  How It Works

1. The user registers a new account by entering a username and password.  
2. The credentials are saved in `localStorage`.  
3. The user then logs in with the registered credentials.  
4. Upon successful login, the user is redirected to the **Dashboard**.  
5. The **Dashboard** checks if the user is authenticated before granting access.  
6. The user can **logout** which clears the login session and redirects to the login page.
